Variable

A variable is used to store a value that will be referenced multiple times.
Variables can be in multiple forms:
- An integer (whole number) e.g. 10
- A string (a single word or sentence) e.g. "Hello!"
- A float (a number with a decimal point) e.g 3.5 - A boolean (a true or false statement) e.g. Programmer = true
