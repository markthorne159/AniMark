Arrays

At first glance arrays can see like an intimidating concept.
However once you get used to using them you'll be glad you did.
So what are arrays? They're essentially a list that can contain a large amount of variables.
Having a single list of variables instead of multiple different single variables can be useful in many different situations.
for example, if you have a bunch of different numbers as variables that all need to be multiplied by a different number
you would have to write multiple repating lines to carry this out.
