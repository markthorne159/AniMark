Python: Indentation

Python has one big difference compared to many other programming languages.
Most languages let you write each line of code one after the other and use any
spacing or patterns you want to depending on how you like to write code.

Python however follows a format which requires different parts of code
to have an 'indent' before them. Pythons indentation enables whichever
program you use to easily understand your code and tell the difference between
'if' statements,functions,'for' loops and any other programming concepts in your code.
