Python: Introduction

Of all of the available programming languages Python has to be one of the simplest to get started with.
This purpose of this tutorial is to introduce you to the basics of Python. This includes installing and setting up Python on your machine,
an introduction to variables and data types in Python, how to structure functions in Python and most importantly, showing you how to write your first program.

Downloading and Installing of Python

Before you can do any sort of programming you need to download and install Python.
Click on the link below to open up Python.org and click on the button highlighted in the image to download the latest version of Python.

Download Python



During the installation you will be presented with a screen that gives you a selection of options to "Customize" your Python installation.
In order to make using Python easier click on the option that says "Add python.exe to Path."
This option will ensure that whenever you want to install new libraries with "pip" Windows will recognise this as a valid command.
Once Python has finished installing it's time to start writing your own code. Create a new folder somewhere on your computer to store your code in and create a new file called "Hello_World.py".
Right click on the file you've created and click on the option that reads "Edit with IDLE".
This option will open up the file in Pythons IDLE editor. Whilst you could just as easily open up and edit the file in something like Notepad or Notepad++ ,however as IDLE
is Pythons main editor it includes a variety of different helpful features as such as syntax highlighting,which makes it easier to understand if your code is written correctly and automatic indentation which is something you will later become incredibly grateful for.

Clicking on the option to open with IDLE will cause a blank window to pop up.


This is the IDLE editor and, more often than not, will be where you write the majority of your code.
Now it's time to write your first piece of code! Enter the folowing lines of code into IDLE and save your file.

print "Hello World!"

Once you've saved your file press "F5" or select "Run" from the options at the top of the screen and click on "Run Module" to run the code you've just written.
Assuming you've written everything correctly you should see "Hello World!" written in your console.


There you have it, you've successfully written your first piece of Python code. It may seem simple at the moment however as you write more code in the future the "print" statement will become one of your most used lines of code with it being perfect for tracking progress through your codes execution or debugging code and fixing errors.
